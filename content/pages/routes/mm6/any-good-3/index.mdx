## Route

### Knight start

- Quick start
- Get the fly scroll
- Use fly scroll on p1
- Swap p4
- Save the game
- Swap p4 back
- Travel to Dragonsand
- Cross to Free Haven (NE)
- Swap p4 twice
- Load the saved game
- Swap p4, p3, p2, p1
- Swap p3
- Travel to Dragonsand
- Loot the chests (need 4000 gold)
- Give crossbow and axe from archer to knight (p4)
- Hire wind master and gate master

### Stables

- TP to Silver Cove
- Rest to reset TP
- Cast fly
- Accept Money quest
- TP to White Cap
- Save near the tree (Alamos)
- Save near the tree (Darkmoor)
- Buy food and rest in the inn 
- Price fix the stables
- Start Winter quest
- Start The Prince of Thieves quest
- TP to Blackshire
- Rest to reset TP
- Cast fly
- Price fix the stables
- Save near the tree (Supreme Temple)
- Get Lord Kilburn's shield
- TP to Free Haven
- Price fix the stables (west)
- Travel to Darkmoor
- Price fix the stables
- Enter Castle Darkmoor
- Get the orb with the Darkmoor save
- TP to Free Haven
- Travel to Silver Cove
- Price fix the stables
- TP to New Sorpigal
- Cast fly
- Save near the trees (Crystals)
- Price fix the stables
- Travel to Castle Ironfist
- Price fix the stables
- Cast fly
- Make a save near the house (Blaster)
- Turn in quests in the castle
- TP to Free Haven
- Start Admission to the Oracle
- Save near the water guild's sign (Sewer)
- Save near the house (VARN)
- Price fix the stables (east)
- Rest 2 days
- Travel to Kriegspire
- Price fix the stables
- Cast fly
- Enter Castle Kriegspire (well; turn around, second turn left)
- Get the orb
- TP to Silver Cove
- Rest to reset TP
- Cast fly
- Turn in Money quest

### Council quests

- TP to Mist
- Rest to reset TP
- Save near the fountain (Forge)
- TP to New Sorpigal
- Travel to Castle Ironfist
- Cast fly
- Start Silvertongue's Refusal
- TP to Silver Cove
- Cast fly
- Use boat to travel to Island North (Cerulean Skies)
- Cast fly
- Enter Castle Alamos
- Get the orb with the Alamos save
- Leave Castle Alamos
- TP to New Sorpigal (fmv)
- Rest to reset TP
- Enter Gharik's Forge
- Get Hourglass with the Forge save
- TP to Mist
- Rest to reset TP
- Cast fly
- Turn in Council Quest
- TP to Free Haven
- Rest to reset TP
- Save near the house (Oracle)
- Enter Sewer
- Use Sewer save to talk to The Prince of Thieves
- TP to White Cap
- Cross to Kriegspire
- Enter Superior Temple of Baa
- Make a save in front of the entrance
- Melee closest to the chest Priest of Baa (High Cleric's Key) * Left sides of doors to clip, return to the entrance with the save
- Melee Priest of Baa (High Sorcerer's Key) * Right sides of doors to clip, return to the entrance with the save
- Open the chest and get the message scroll, return to the entrance with the save
- Leave Superior Temple of Baa
- Talk to The Hermit on the Mountain
- Enter Devil Outpost
- Melee Devil of Baa and get the plans
- TP to White Cap
- Turn in Winter quest
- Turn in The Prince of Thieves quest
- TP to Free Haven
- Rest to reset TP
- Cast fly
- Turn in Admission to the Oracle (press Esc to skip the cutscene)
- Turn in Council Quest (Osric Temper)

### The Oracle

- TP to New Sorpigal
- Travel to Castle Ironfist
- Travel to Hermit's Isle
- Enter Supreme Temple of Baa
- Get the orb with the Supreme save
- Rest to reset TP
- Cast fly
- Travel to Dragonsand
- Enter Tomb of VARN
- Get the control cube using the VARN save (click left, top, right)
- TP to Free Haven
- Rest to reset TP
- Enter Oracle
- Power on
- Place the orbs using the Crystals save
- Talk to The Oracle using the Oracle save (Control Center, Kreegan topics)
- Talk to The Oracle using the Oracle save (enter Control Center)
- Get a blaster rifle from the chest with the Blaster save and equip the Knight (p4) with it
- TP to New Sorpigal
- Travel to Castle Ironfist
- Start Boredom quest
- Start The Third Eye quest
- Cure the Knight if there are any status effects
- Release Archibald
- TP to Blackshire
- Buy food
- Travel to Sweet Water (fmv)
- Enter The Hive
- Kill Demon Queen
