This mod will eradicate all but 1 party member allowing to quickly setup a solo playthrough. Is intended for 1.0 - 1.1 versions of the game.

[Download](/mods/mm7/mm7-solo-mod-vanilla.zip)

![Solo mod for Might & Magic VII](./mm7-solo-mod-vanilla.jpg)

It is implemented via altered EVT script for the starting map.
